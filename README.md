# Devcamper Api
> the backend server for the devcamper application

## Install Dependencies

```
npm i 
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm run start
```

## Database Seeder

To seed the database with users, bootcamps, courses and reviews with data from the "\_data" folder, run

```
# to import all data
node seeder -i

# to delete all data
node seeder -d
```
